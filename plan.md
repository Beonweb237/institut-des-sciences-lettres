# Plan — Template Université & Grande École (Secteur 18)

## Contexte
Construction d'un template web premium pour universités, grandes écoles, écoles d'ingénieurs et écoles de commerce. Le template doit être institutionnel, prestigieux, inspirant et orienté conversion (recrutement étudiant).

## Phase 1: Analyse & Conception (Orchestrator)
- Charger les deux fichiers secteur (uploadés)
- Synthétiser les besoins métiers, personas et parcours utilisateurs
- Définir l'architecture des pages et les composants réutilisables

## Phase 2: Design System (Orchestrator)
- Créer le système de design (couleurs, typographie, espacement)
- Définir les composants partagés
- Planifier les assets visuels nécessaires

## Phase 3: Génération d'Assets (Sub-agents parallèles)
- Générer les images du template (campus, étudiants, portraits alumni, recherche)
- Optimiser et préparer les assets

## Phase 4: Développement (vibecoding-webapp-swarm)
- **Stage 4a**: Scaffold projet Next.js + Tailwind + shadcn/ui
- **Stage 4b**: Développement des pages publiques (Accueil, Formations, Admission, Recherche, Campus, Vie Étudiante, Entreprises, Alumni, International, À Propos, Contact)
- **Stage 4c**: Développement de l'Espace Étudiant/Candidat (User Space)
- **Stage 4d**: Développement de l'Espace Administration (Admin Space)
- **Stage 4e**: Polish final, SEO, accessibilité, déploiement

## Livrables
- Site web complet déployé avec toutes les pages publiques
- Espace utilisateur (étudiant/candidat)
- Espace administration
- Design system documenté
